# Python Video Generator

A simple Python project for generating vertical videos using MoviePy.

This project creates a 9:16 video (720x1280) with text in the center of the screen.
It is suitable for Instagram Reels, TikTok, and YouTube Shorts.

---

## Features

- Vertical video format (9:16)
- Custom text overlay
- Adjustable duration
- MP4 output
- Easy to modify

---

## Requirements

- Python 3.8 or higher
- FFmpeg (installed automatically by MoviePy)

---

## Installation

First, install the required libraries:

pip install -r requirements.txt

---

## Usage

Run the main script:

python main.py

After running the script, a video file named `output_video.mp4`
will be generated in the project directory.

---

## Project Structure

python-video-generator/

---

## Customization

You can edit the following values in `main.py`:

- Text content
- Font size
- Video duration
- Background color
- Video resolution
