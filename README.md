Instagram Reels, TikTok, and YouTube Shorts.

## Description

This project generates a vertical (9:16) MP4 video with:
- Custom text
- Adjustable duration
- Configurable resolution
- Clean and simple structure

The main goal is to provide a minimal Python-based video generator
that can be easily extended.
